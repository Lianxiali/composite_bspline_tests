Switch the version of IFED by changing the parameters in __IBFEMethod__ section  

[1] elemental IFED  
    ```use_consistent_mass_matrix = TRUE```   
    
[2] nodal IFED  
    <code>use_consistent_mass_matrix = FALSE  <br/>
    IB_use_nodal_quadrature    = TRUE<code>
